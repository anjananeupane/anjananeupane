Block-Level Elements

Block-level elements typically take up the full width available and start on a new line. In this HTML document, the block-level elements are:

  • header: Styled with a background color (#333), text color (#fff), centered text alignment, and padding (1em 0). It has a blue border and margin of 10px all around as per the CSS rule for block-level elements.

  • .container: Has a maximum width of 1200px, centered with auto margins (margin: 0 auto), background color (#fff), padding (20px), and a box-shadow. It also has a blue border and 10px margin.
  • .intro: Margin-bottom of 20px for spacing between sections. Blue border and 10px margin.
  • section: Each section has a blue border and 10px margin.
  • p: Paragraphs have a blue border and 10px margin.
  • h1: The h1 inside the header and section have a blue border and 10px margin. The h1 in the section also has an inline style that changes its display to inline with a right margin of 10px.

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. In this HTML document, the inline elements are:

  • a: Anchor links (<a>) are styled to have a blue text color (#1a73e8) and no text decoration initially, but underlined on hover. Each a element has a red border.
  • span: Spans (<span>) are used for inline text elements and have a red border.

Margins

Block-level elements have a 10px margin all around, which provides spacing outside the border.

The .intro class has an additional bottom margin of 20px to create space between sections.

The inline h1 in the section has an inline style with a margin-right of 10px.

Padding

The body element has zero margin and padding, ensuring the content starts at the edge of the browser window.

header and footer have 1em 0 padding, creating space inside the element above and below the text.

The main element has 40px padding, providing space inside the element around its content.

The .container has 20px padding, creating space inside the element around its content.

Summary

Block-level elements are highlighted with blue borders and include elements like header, footer, main, .container, section, p, and h1.

Inline elements are highlighted with red borders and include <a> and <span>.

Margins provide spacing outside elements, while padding creates space inside elements.
